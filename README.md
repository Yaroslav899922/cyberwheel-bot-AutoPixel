# 🏎️ CyberWheel AI Engine (v2.5 Cloud Edition)

**CyberWheel** — це повністю автономна інтелектуальна система кураторства контенту. Бот самостійно моніторить провідні світові та українські медіа, аналізує новини за допомогою штучного інтелекту та створює експертні дайджести для Telegram-каналу. 

Головна дійова особа системи — **Агент Софія**, віртуальний ШІ-аналітик, який адаптує контент, робить професійні висновки та взаємодіє з аудиторією.

---

## 🎯 Мета проекту (The Vision)
Створити безперервний конвеєр обробки даних, який перетворює "сирий" потік новин на структурований, візуально досконалий та корисний контент для бізнесу (автосалону) без участі людини.

## 🛠️ Технологічний стек
* **Мова:** Python 3.11+
* **ШІ:** Google Gemini API (модель Flash-Lite)
* **База даних:** Upstash Redis (інтеграція через нативний `urllib` POST)
* **Фреймворки:** Flask (багатопотоковий веб-сервер)
* **Інструменти:** Feedparser, Newspaper3k, Schedule, Регулярні вирази (re)
* **Хостинг:** Render.com + GitHub

## 📜 Історія еволюції проекту (Changelog)

### v2.5 (Cloud Edition) — *Поточна версія*
* **Перехід у хмару:** Локальний файл пам'яті замінено на хмарну базу **Upstash Redis**.
* **Fix дублікатів:** Переписано логіку звернення до бази на POST-запити (формат JSON), що дозволило зберігати повні URL-адреси без розрізання та повністю усунуло проблему появи дублікатів після перезавантаження сервера.
* **Smart Sleep:** Впроваджено функцію розумного сну. Бот автоматично призупиняє пошук новин рівно о 22:00, публікує побажання "На добраніч" і йде в режим очікування до 10:00.
* **Глобальні джерела:** Змінено вектор контенту. Локальні агрегатори замінені на топові світові першоджерела (США, Велика Британія, Китай).

### v2.0 (AI Edition)
* Інтеграція Google Gemini API для генерації саммарі та перекладу текстів.
* Створення блоку "Вердикт Агента Софії" — аналітичного висновку до кожної новини.
* Впровадження інтерактивних кнопок ("Читати повністю") та клікабельних посилань на соцмережі автоцентру.
* Налаштування суворої HTML-розмітки для захисту від помилок Telegram API.
* Автоматичне підтягування головного фото (image_url) над текстом посту.

### v1.0 (MVP)
* Базовий парсинг RSS-стрічок через `feedparser`.
* Локальне збереження оброблених посилань у файл `parsed_urls.txt`.
* Розгортання базового фонового Flask-сервера для підтримки життєздатності (keep-alive) на платформі Render.
