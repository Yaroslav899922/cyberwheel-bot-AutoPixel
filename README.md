# 🏎️ CyberWheel AI Engine (v1.0)

**CyberWheel** — це повністю автономна інтелектуальна система кураторства контенту. Бот самостійно моніторить провідні світові та українські медіа, аналізує новини за допомогою штучного інтелекту та створює експертні дайджести для Telegram.

---

## 🎯 Мета проекту (The Vision)
Створити безперервний конвеєр обробки даних, який перетворює "сирий" потік новин на структурований, цікавий та корисний контент без участі людини.

---

## 🏗️ Архітектура системи (Audit Trail)

| Етап | Що планували | Як реалізовано | Статус |
| :--- | :--- | :--- | :--- |
| **🔍 Збір (Scraping)** | Отримання чистого тексту з веб-сторінок | Бібліотека `newspaper3k` + `requests` з імітацією браузера (User-Agent) | ✅ Готово |
| **🛰️ Моніторинг (Scout)** | Автоматичне стеження за новими статтями | Розумний **RSS-парсер** (`feedparser`) з гнучким списком джерел у `sources.txt` | ✅ Готово |
| **🧠 Аналіз (Brain)** | Використання ШІ для створення дайджестів | Інтеграція **Gemini 3.1 Pro/Flash** (найновіші SOTA моделі від Google) | ✅ Готово |
| **💾 Пам'ять (DB)** | Захист від повторних публікацій | Система локального логування оброблених посилань у `parsed_urls.txt` | ✅ Готово |
| **📲 Доставка (Bot)** | Відправка контенту користувачу | Власний **Telegram Bot** з підтримкою Markdown-розмітки | ✅ Готово |
| **☁️ Хмара (Cloud)** | Автономна робота 24/7 | Деплой на платформу **Render** + **Cron-job** сервіс для підтримки активності | ✅ Готово |
| **🛡️ Безпека (Sec)** | Захист конфіденційних даних | Архітектура прихованих ключів через `.env` та **Environment Variables** | ✅ Готово |

---

## 🛠️ Технологічний стек
*   **Мова:** Python 3.11+
*   **ШІ:** Google Gemini 3.1 API (Pro & Flash)
*   **Фреймворки:** Flask (для веб-сервера), Aiogram/Requests (для Telegram)
*   **Інструменти:** Newspaper3k, Feedparser, Python-dotenv
*   **Хостинг:** Render.com + GitHub
