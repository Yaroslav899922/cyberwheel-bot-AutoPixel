# 🏎️ CyberWheel AI Engine (v2.0)

**CyberWheel** — це повністю автономна інтелектуальна система кураторства контенту. Бот самостійно моніторить провідні світові та українські медіа, аналізує новини за допомогою штучного інтелекту та створює експертні дайджести для Telegram-каналу. 

Головна дійова особа системи — **Агент Софія**, віртуальний ШІ-аналітик, який адаптує контент, робить професійні висновки та взаємодіє з аудиторією.

---

## 🎯 Мета проекту (The Vision)
Створити безперервний конвеєр обробки даних, який перетворює "сирий" потік новин на структурований, візуально досконалий та корисний контент для бізнесу (автосалону) без участі людини.

---

## 🏗️ Архітектура системи (Audit Trail)

| Етап | Що планували | Як реалізовано | Статус |
| :--- | :--- | :--- | :--- |
| **🔍 Збір (Scraping)** | Отримання чистого тексту та фото | Бібліотека `newspaper3k` + `requests` з імітацією браузера (User-Agent) | ✅ Готово |
| **🛰️ Моніторинг (Scout)** | Автоматичне стеження за статтями | Розумний **RSS-парсер** (`feedparser`) з розкладом 30/120 хв | ✅ Готово |
| **🧠 Аналіз (Brain)** | ШІ-дайджести та переклад | Інтеграція **Gemini 3.1 Pro / 2.0 Flash** з жорстким HTML-форматуванням | ✅ Готово |
| **🎨 Контент-дизайн** | Ідеальна подача в Telegram | Великі фото (link_preview_options), Inline-кнопки, підвал із соцмережами | ✅ Готово |
| **🎭 Персоналізація**| "Живий" характер бота | Логіка ранкових вітань (о 8:00) та авторський блок "Вердикт Агента Софії" | ✅ Готово |
| **💾 Пам'ять (DB)** | Захист від дублів | Система локального логування оброблених посилань у `parsed_urls.txt` | ✅ Готово |
| **☁️ Хмара (Cloud)** | Автономна робота 24/7 | Деплой на **Render** + фоновий Flask-сервер + **Cron-job** (будильник) | ✅ Готово |
| **🛡️ Безпека (Sec)** | Захист API-ключів | Архітектура прихованих ключів через `.env` та **Environment Variables** | ✅ Готово |

---

## 🛠️ Технологічний стек
*   **Мова:** Python 3.11+
*   **ШІ:** Google Gemini 3.1 API (Pro & Flash-Lite)
*   **Фреймворки:** Flask (для багатопотокового веб-сервера), Requests (для Telegram API)
*   **Інструменти:** Newspaper3k, Feedparser, Python-dotenv, Регулярні вирази (re)
*   **Хостинг:** Render.com + GitHub
